# Lista1_GuilhermeLeal

## Instruções

```
 pip install requerimentos.txt
 python main.py -l X -t Y
```
 X = tamanho do vetor (1-1000)

 Y = BubbleSort ou InsertionSort ou MergeSort ou SelectionSort

 Exemplo
 ```
    python main.py -l 50 -t BubbleSort
 ```
